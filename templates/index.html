CryptoAnalyzer

Analizador de Fortaleza Criptográfica

Proyecto 14

Objetivo

Crear una herramienta que analice la fortaleza de diferentes algoritmos criptográficos, realice criptoanálisis de cifrados clásicos y evalúe la seguridad de implementaciones modernas.

Tecnologías sugeridas

  • Lenguaje: Python
  • Criptografía: PyCryptodome, cryptography
  • Análisis: NumPy para análisis estadístico
  • Interfaz: Flask web app o PyQt5
  • Visualización: Matplotlib para frecuencias

Requisitos Funcionales

  • 1. Implementación de cifrados clásicos (César, Vigenère, Playfair).
  • 2. Herramientas de criptoanálisis: análisis de frecuencia, fuerza bruta.
  • 3. Evaluación de fortaleza de algoritmos modernos (AES, RSA).
  • 4. Detección de debilidades en implementaciones (padding, IV reutilización).
  • 5. Comparación de rendimiento entre diferentes algoritmos.
  • 6. Generación de reportes de auditoría criptográfica con recomendaciones.
